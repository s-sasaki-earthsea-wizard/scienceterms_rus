# 物理用語 ロシア語クイズ

物理用語のロシア語学習用クイズアプリケーションです。高校物理から大学1-2年生レベルの物理用語100問を収録しています。

## 機能

- **2つの出題モード**
  - 日本語 → ロシア語
  - ロシア語 → 日本語
- **10問のクイズ形式**
  - 同一ゲーム内で重複なし
  - 正解/不正解のフィードバック
  - 結果画面で正答率と間違えた問題を表示
- **100問の物理用語辞書**
  - 力学、熱力学、電磁気学、量子力学などをカバー

## 開発環境のセットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview
```

## GitHub Pagesへのデプロイ

```bash
# ビルドとデプロイ
npm run deploy
```
